About This App

Django Authentication Web App

This is a complete authentication system built with Django, featuring:

  • User Registration with email validation
  • Secure Login/Logout functionality
  • User Profile management
  • Profile picture upload
  • Protected dashboard for authenticated users
  • Responsive design with Bootstrap 5
  • PostgreSQL database ready for Azure deployment
Technologies Used
  • Django 4.2 - Python Web Framework
  • Bootstrap 5 - Frontend Framework
  • PostgreSQL - Database
  • Azure App Service - Hosting
Features Demonstrated
  • User authentication system
  • One-to-one relationship for user profiles
  • File upload for profile pictures
  • Form validation and error handling
  • Message framework for user notifications
  • Template inheritance and reusable components
Ready for Azure: This app is configured with environment variables and ready to be deployed to Azure App Service with PostgreSQL database.
